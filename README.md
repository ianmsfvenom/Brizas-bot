# 🤔 ¿Qué es?

El bot de Bitroi es un bot basado en [MhankBars] (https://github.com/MhankBarBar/termux-wabot) con la API principal de wpp web creada por [adiwajshing] (https://github.com/adiwajshing/ Baileys) y su equipo. El bot fue modificado y agregó varios comandos que fueron desarrollados principalmente para satisfacer al pueblo brasileño, con generador de ciudadanos, CPF, CC, y mucho más.

---


# ⬇️ Instalación

[Termux](https://play.google.com/store/apps/details?id=com.termux&hl=pt_BR&gl=US)

```
git clone https://github.com/Axel-bytu/Bitroi
cd Bitroi
bash install.sh
npm install

(iniciar o bot manualmente)
npm start

(reiniciar o bot automaticamente em caso de crash)
sh start.sh

(ou)
pm2 start index.js

```
---
# 🙋‍♂️ Cómo cambiar la configuración del bot
```
cd
cd Bitroi
apt install nano
nano config.js
```
